Τι κάνει ο κώδικας ds_demo.c
	1.	Δυναμικός πίνακας (array)
	•	Δεσμεύει μνήμη για N = 100000 ακέραιους.
	•	Γεμίζει τον πίνακα με τιμές i * 3.
👉 Αυτό παράγει WRITE σε συνεχόμενη μνήμη.
	•	Διαβάζει κάθε 64ο στοιχείο.
👉 Αυτό παράγει READ με σχεδόν γραμμικό pattern.
	2.	Συνδεδεμένη λίστα (linked list)
	•	Δημιουργεί μια λίστα με M = 20000 κόμβους, με malloc για κάθε κόμβο.
👉 Οι διευθύνσεις μνήμης θα είναι διάσπαρτες στο heap.
	•	Κάνει διάσχιση της λίστας και αθροίζει τις τιμές.
👉 Αυτό παράγει READ σε μη συνεχόμενη μνήμη.
	•	Τροποποιεί τιμές σε μερικούς κόμβους (κάθε 1024 περίπου).
👉 Αυτό παράγει WRITE σε διάσπαρτες διευθύνσεις.
	3.	Τυχαίες προσπελάσεις στον πίνακα
	•	Με έναν απλό ψευδο-τυχαίο τύπο (mod) επιλέγει δείκτες και κάνει:
	•	READ από array.
	•	WRITE σε array (με XOR).
	•	Αυτό σπάει τη γραμμικότητα του array και δείχνει πιο «τυχαίο» access pattern.
	4.	Καθαρισμός
	•	Ελευθερώνει (free) όλα τα nodes της λίστας και τον πίνακα.
👉 Και αυτό γράφει/διαβάζει στο heap (metadata του malloc/free).
	5.	Εκτύπωση
	•	Στο τέλος τυπώνει:  Done. Sink=XXXX
    (το Sink είναι απλώς ένας μετρητής για να αναγκάσει τον compiler να κρατήσει τις προσπελάσεις).

    compile: gcc -O0 -g -o DataStructuresCode DataStructuresCode.c
    run: ./DataStructuresCode
